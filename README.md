PERANCANGAN DAN IMPLEMENTASI BASIS DATA SISTEM PEMESANAN KOPI KENANGAN

Gambaran Umum Sistem
Sistem Pemesanan Kopi Kenangan adalah sistem basis data yang dirancang untuk mengelola data transaksi pemesanan kopi secara terstruktur dan terintegrasi. Sistem ini mencakup pengelolaan data pelanggan, produk kopi, serta transaksi pemesanan dan pembayaran.
Perancangan basis data dilakukan menggunakan konsep basis data relasional dengan menerapkan normalisasi hingga bentuk normal ketiga (3NF) guna meminimalkan redundansi data dan menjaga konsistensi antar tabel. Struktur data divisualisasikan melalui Entity Relationship Diagram (ERD) sebagai acuan dalam implementasi basis data.
Sistem ini mendukung pengolahan data menggunakan query SQL untuk kebutuhan pencatatan transaksi, pengambilan data terintegrasi, serta penyajian informasi dalam bentuk rekapitulasi.

Tujuan Sistem
1. Mengelola data pemesanan kopi secara terstruktur
2. Menerapkan normalisasi data hingga 3NF
3. Menyusun ERD sebagai model relasi data
4. Mengimplementasikan basis data menggunakan MySQL
5. Menguji query SQL (DDL, DML, TCL, JOIN, GROUP BY, HAVING)

Ruang Lingkup Sistem
1. Data pelanggan dan produk kopi
2. Transaksi pemesanan dan detail transaksi
3. Pengolahan data menggunakan SQL
4. Tidak mencakup pengembangan antarmuka aplikasi

Tools yang Digunakan
1. DBMS : MySQL
2. Bahasa Query : SQL
3. Tools Pendukung : MySQL Workbench, phpMyAdmin, XAMPP, Text Editor
